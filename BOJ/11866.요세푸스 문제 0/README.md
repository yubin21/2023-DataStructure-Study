## 11866: 요세푸스 문제 0
### 문제와 관련된 개념
큐 관련 개념은 우측 참조. [2164: 카드](https://github.com/yubin21/2023-DataStructure-Study/tree/main/BOJ/2164.%EC%B9%B4%EB%93%9C)
### 문제 풀이
num을 받아 그 num이 0인 경우 pop, else인 경우 push를 한다. for문을 통해 stack에 있는 모든 값을 더하여 이를 출력한다.
